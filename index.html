Tentang Saya

Saya adalah mahasiswa Universitas Pembangunan Nasional “Veteran” Jawa Timur dari program studi Sistem Informasi dengan minat utama di bidang Full Stack Development. Saya memiliki dasar yang kuat dalam pengembangan Front End serta ketertarikan pada UI/UX Design, dan saat ini sedang memperdalam kemampuan Back End untuk menjadi developer yang menyeluruh. Saya memiliki keahlian menengah dalam pemrograman dan desain grafis, serta mampu bekerja sama dalam tim dan mengelola waktu secara efektif. Dengan semangat belajar yang tinggi, saya berkomitmen untuk terus mengembangkan keterampilan saya agar menjadi Full Stack Developer yang profesional dan andal

Keahlian saya meliputi:

  • Bahasa pemrograman: PHP, Java, JavaScript, HTML, CSS
  • Framework: Laravel, Tailwind CSS
  • Database: MySQL
  • Tools: Visual Studio Code, XAMPP, Laragon, Figma, Canva, Git

Proyek Pilihan

BacaPoint

Aplikasi ini berfungsi sebagai platform literasi digital yang mendorong kebiasaan membaca secara konsisten dan menyenangkan melalui sistem poin sebagai insentif utama.

PrintaGo

Printago adalah aplikasi pengiriman file secara online ke tempat percetakan. Dengan aplikasi ini, pengguna bisa mengunggah file yang ingin dicetak, memilih jenis cetakan, dan menentukan lokasi percetakan. Setelah itu, file akan langsung diteruskan ke percetakan, sehingga pengguna bisa mengambil hasil cetak tanpa harus datang dua kali.
